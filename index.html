📅 予約管理 (Phase 2 で実装予定)

現在は月次の予約「数」のみ集計しています。本機能では下記を追加する想定です：

  • 個別予約レコード（日時 / 顧客名 / 施術メニュー / 担当 / ステータス）
  • カレンダー表示・日別キャパシティ管理
  • キャンセル率・無断キャンセル率の自動算出
  • LINE公式 / Googleカレンダー / RESERVA など外部予約システムとのCSV連携
  • 顧客カルテ（来院履歴・LTV・初回来院チャネル）
次のステップ: 既存の予約システム（例: RESERVA、Airリザーブ、LINE公式）からCSVを書き出せるか確認できると、すぐにこの画面で取り込めるよう実装できます。
💡 このサンプルでできること: 月ごとの売上 / 広告費 / 新規 / リピートを入力すると、CPA・ROAS・推移グラフを自動算出します。データはブラウザの localStorage に保存され、エクスポート/インポートでバックアップ可能です。
本番運用するときは Supabase / Firebase などのDBに繋ぎ替えれば、複数端末から同じデータを見られるようになります。
